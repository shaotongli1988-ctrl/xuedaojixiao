## 维护规则

1. 同一规则只能在一份“唯一事实源”文档里定义。
2. 其他文档引用该规则时，只写链接和引用说明，不重复抄写完整内容。
3. 如果规则发生变化，先更新对应唯一事实源，再同步回写受影响文档。
4. 仓库根目录下的其他教程、示例和旧模块文档，不再作为本项目入口。
