## 实现边界

1. 本轮只允许实现：
   - `purchaseOrder page / info / add / update / delete`
   - `supplier page / info / add / update / delete`
2. 本轮必须遵守：
   - 采购订单状态：`draft / active / cancelled`
   - 供应商状态：`active / inactive`
   - 采购订单删除只允许 `draft`
   - 供应商删除只允许 `inactive` 且无关联有效采购订单
3. 本轮权限边界固定为：
   - `HR` 全量管理采购订单和供应商
   - 部门经理只允许在部门树范围内管理采购订单，对供应商只读
   - 员工无任何主题11后端入口
4. 本轮不允许：
   - 改共享基础层
   - 扩到采购计划、询价管理、审批、收货 / 入库、对账、付款、库存联动、采购报表
   - 用聊天结论替代已冻结文档
