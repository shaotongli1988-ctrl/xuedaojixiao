1. 主题名称固定为：`文件管理与知识库`
2. 当前阶段0先保持一个合并主题，不拆成“文件管理 / 知识库”两个独立主题。
3. 首批范围固定为：

   - 知识条目分页 / 统计 / 新增 / 编辑 / 删除
   - 知识图谱
   - 知识搜索
   - 百问百答列表 / 新增

6. 主题21 独立归入业务域“行政协同”，不并入培训、合同、采购、资产或系统管理域。
