## 最小测试矩阵

### 正常路径

1. `HR` / 部门经理可新增招聘计划
2. `HR` / 部门经理可导入招聘计划并生成草稿
3. `HR` / 部门经理可删除草稿计划
4. `HR` / 部门经理可编辑草稿计划
5. `HR` / 部门经理可查看详情
6. `HR` / 部门经理可提交草稿计划备案生效
7. `HR` / 部门经理可关闭生效计划
8. `HR` / 部门经理可作废生效计划
9. `HR` / 部门经理可重新开启 `closed / voided` 计划
10. `HR` 可导出招聘计划摘要

### 异常路径

1. 员工访问主题16接口被拒绝
2. 范围外经理访问或操作被拒绝
3. 非法状态下执行 `delete / submit / close / void / reopen` 被拒绝
4. 部门经理执行 `export` 被拒绝

### 边界路径

1. `draft` 可编辑
2. `draft` 可删除但不可关闭、作废或重新开启
3. `active` 不可回退为 `draft`，不可编辑或删除
4. `closed / voided` 仅可重新开启，不可编辑、删除或再次提交

### 权限路径

1. 部门经理仅可操作 `targetDepartmentId` 落在本人部门树范围内的招聘计划
2. 部门经理无导出权限
3. 员工无菜单、无详情、无写动作

### 数据边界

1. 详情与导出都不返回简历全文、候选人联系方式、面试评语全文、录用决策全文和预算明细对象
