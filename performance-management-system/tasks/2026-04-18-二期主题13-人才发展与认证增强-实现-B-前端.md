## 范围

1. 实现主题13前端页面：
   - 能力模型列表
   - 能力项详情
   - 员工能力画像摘要
   - 证书列表
   - 证书详情
   - 发放记录 / 获得记录
2. 前端只对接冻结允许的：
   - `capabilityModel page / info / add / update`
   - `capabilityItem info`
   - `capabilityPortrait info`
   - `certificate page / info / add / update / issue / recordPage`
3. `HR` 可维护能力模型与证书，并可发放证书。
4. 部门经理只读能力模型、能力项、画像摘要、证书定义与证书记录。
5. 员工无入口。

## 非目标

1. 不改后端 entity / service / controller
2. 不改 `menu.json`
3. 不做课程主链、课程学习过程、课程结业自动发证
4. 不做人才资产、简历池、面试流程、合作伙伴
5. 不做移动端
6. 不展示人才档案全文、简历全文、面试评语全文、证书附件或扫描件

## 验收口径

1. 页面资源按冻结边界拆分为：
   - `capability`
   - `certificate`
2. `HR` 可走通能力模型列表 / 详情 / 创建 / 编辑、证书列表 / 详情 / 创建 / 编辑 / 发放 / 记录查询。
3. 部门经理只读，无新增 / 编辑 / 发放入口。
4. 员工无页面入口。
5. 前端必须通过：
   - `corepack pnpm run type-check`
   - `corepack pnpm run build`
