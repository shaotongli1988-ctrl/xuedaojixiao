## 已冻结资源边界

1. 主题11窗口B允许在同一冻结包内定义以下资源：
   - `purchaseOrder`
   - `supplier`
   - `purchaseReport`
2. `purchaseOrder` 接口固定为：
   - `page`
   - `info`
   - `add`
   - `update`
   - `delete`
   - `submitInquiry`
   - `submitApproval`
   - `approve`
   - `reject`
   - `receive`
   - `close`
3. `supplier` 接口固定为：
   - `page`
   - `info`
   - `add`
   - `update`
   - `delete`
4. `purchaseReport` 接口固定为：
   - `summary`
   - `trend`
   - `supplierStats`
5. 状态固定为：
   - 采购订单：`draft / inquiring / pendingApproval / approved / received / closed / cancelled`
   - 供应商：`active / inactive`
6. 权限固定为：
   - `HR` 全量可见、可建、可改、可删、可执行流程动作、可看报表
   - 部门经理仅可在部门树范围内管理采购订单、执行询价/审批/收货/关闭，并对供应商只读
   - 员工完全无入口

## 数据与脱敏边界

1. 采购订单金额 `totalAmount`：
   - `HR` 全量可见
   - 部门经理仅在本人部门树范围内可见
   - 员工不返回
2. 采购增强详情允许增加以下最小流程字段：
   - `expectedDeliveryDate`
   - `approvedBy`
   - `approvedAt`
   - `approvalRemark`
   - `closedReason`
   - `receivedQuantity`
   - `receivedAt`
   - `items`
   - `inquiryRecords`
   - `approvalLogs`
   - `receiptRecords`
3. 供应商敏感字段：
   - `bankAccount`
   - `taxNo`
   - `contactName`
   - `contactPhone`
   - `contactEmail`
4. 上述供应商敏感字段规则：
   - `HR` 返回完整值
   - 部门经理仅返回脱敏摘要
   - 员工不返回
