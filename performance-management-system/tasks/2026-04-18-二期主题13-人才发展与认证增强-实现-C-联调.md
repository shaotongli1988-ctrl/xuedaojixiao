   - 真实现象：`capabilityModel:page` 与 `certificate:page` 返回 `404 Not Found`，而既有主题接口仍正常。
   - 结论：8006 端口上的同仓库实例未加载当前最新实现，属于本地旧实例漂移，不是主题13代码缺失。

   - 结果：8013 新实例成功监听，日志显示 Midway 已启动。
5. 首次 smoke 指向 `http://127.0.0.1:8013`：发现 1 个脚本误报。
   - 真实现象：`hr_admin capabilityPortrait:info` 因证书记录按 `issuedAt DESC` 返回首条为 `销售员工`，而脚本硬编码使用第一页第一条记录去断言画像员工，导致误报。
   - 处理：仅修正 `smoke-stage2-performance.mjs` 的断言方式，改为优先匹配期望员工名称，不改任何业务实现。

   - 真实结果：`PASS: 82 / FAIL: 0 / SKIP: 0 / Conclusion: PASSED`
