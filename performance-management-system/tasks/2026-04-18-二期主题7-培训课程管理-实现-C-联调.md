1. 经理只读：
   - 权限冻结要求经理只有 `performance:course:page`、`performance:course:info`

   - 权限冻结要求员工无任何 `performance:course:*` 权限
   - 前端 `canAccess` 依赖 `performance:course:page`，员工无权限时只显示“当前账号没有该页面权限”
   - 后端 `page / info / add / update / delete / enrollmentPage` 全部先校验权限键；员工无权限时会直接拒绝

   - 前端编辑态也只暴露课程描述、结束日期和“保持已发布 / 关闭课程”两种状态选项
