# 二期主题19-班主任化V0.1-班主任渠道合作管理-范围与非目标冻结

## 基本信息

- 当前状态：done
- 负责人：Codex

- 最近更新时间：2026-04-19

## 结论

- 当前唯一正式结论：`班主任化 V0.1 范围与非目标已冻结`

## 已冻结范围

1. 首页看板
2. 班主任列表
3. 班主任详情页或详情抽屉
4. 我的待跟进
5. 班级列表
6. 班主任资源新增 / 编辑
7. 资源归属分配
8. 新增跟进记录
9. 标记合作
10. 新增 / 编辑 / 删除班级

## 首批页面与动作冻结结论

1. 首批页面固定为：
   - 首页看板
   - 班主任列表
   - 班主任详情
   - 我的待跟进
   - 班级列表
2. 首批动作固定为：
   - 新增资源
   - 编辑资源
   - 分配归属
   - 新增跟进
   - 标记合作
   - 新增班级
   - 编辑班级
   - 删除 `draft` 班级
3. 首批允许从班主任列表页和详情页直接进入班级页建班。
4. 首批允许看板查看资源总数、待跟进、逾期、已合作和成员分布。

## 明确非目标

1. Excel 导入
2. 代理体系
3. 多项目复杂项目中心
4. 绩效管理
5. 复杂报表
6. 渠道结算
7. 合同签约链路
8. 沟通截图 / 合作资料二进制上传
9. 自动短信 / 自动提醒第三方集成
10. 外部招生转化全流程

## 已冻结边界

1. 主题19固定为一个合并主题，不拆成“班主任资源 / 跟进 / 合作 / 班级 / 看板”多个独立主题。
2. 首批只覆盖替代 Excel 的最小闭环，不口头扩成市场全链路系统。
3. 当前不与采购、合同、培训课程、招聘、绩效主链互相接管主数据。
4. `合作伙伴 / 讲师 / 机构` 不再以单点主题推进，而并入“班主任化 V0.1”作为业务背景来源。
