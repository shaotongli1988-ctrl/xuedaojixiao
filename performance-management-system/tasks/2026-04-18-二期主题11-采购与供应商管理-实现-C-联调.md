## 联调边界

1. 本轮联调只验证：
   - 采购订单 `page / info / add / update / delete`
   - 供应商 `page / info / add / update / delete`
2. 本轮必须覆盖：
   - `hr_admin`
   - `manager_rd`
   - `employee_platform`
3. 本轮必须验证：
   - 经理采购订单部门树范围管理
   - 经理供应商只读与敏感字段脱敏
   - 员工无页面 / 无接口 / 无数据返回
   - 采购订单 `draft` 删除限制
   - 供应商 `inactive + 无关联有效采购订单` 删除限制
4. 本轮不允许：
   - 把审批、收货 / 入库、对账、付款、库存联动拉进联调范围
   - 在没有真实运行证据时口头宣称联调完成
