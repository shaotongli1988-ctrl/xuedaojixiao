# 二期主题21-文件管理与知识库-范围与非目标冻结

## 基本信息

- 当前状态：done
- 负责人：Codex
- 创建时间：2026-04-19
- 最近更新时间：2026-04-19

## 结论

- 当前唯一正式结论：`文件管理与知识库范围与非目标已冻结`

## 已冻结范围

1. 文件管理
2. 知识库
3. 知识图谱
4. 知识搜索
5. 百问百答元数据

## 首批页面与动作冻结结论

1. 首批页面固定为：
   - 文件管理
   - 知识库
2. 首批动作固定为：
   - 文件分页 / 详情 / 统计 / 新增 / 编辑 / 删除
   - 知识条目分页 / 统计 / 新增 / 编辑 / 删除
   - 知识图谱查看
   - 知识关键词搜索
   - 百问百答列表 / 新增
3. 文件管理首批允许按 `status / category / confidentiality / storage / keyword` 过滤。
4. 知识库首批允许按 `status / category / tag / keyword` 过滤。

## 明确非目标

1. 文件目录树
2. 权限继承
3. 二进制文件存储服务
4. 正文富文本编辑器
5. AI 问答 / RAG / 模型推理
6. 员工自助知识门户
7. 年检材料、荣誉管理、宣传资料、美工协同、快递协同、车辆管理、知识产权管理

## 已冻结边界

1. 主题21固定为一个合并主题，不拆成“文件管理 / 知识库”两个独立主题。
2. `documentCenter` 只承接文件元数据台账，不负责真实文件物理存储。
3. `knowledgeBase` 只承接知识条目元数据、图谱、搜索和百问百答元数据，不负责正文全文和 AI 生成答案。
4. 当前不触碰主题 `1-20` 已关闭结论，也不反向改写既有业务域归属。
