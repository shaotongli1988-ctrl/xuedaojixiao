## 已冻结资源边界

1. 资源名固定为：
   - `documentCenter`
   - `knowledgeBase`
2. 首批接口固定为：
   - `documentCenter page / info / stats / add / update / delete`
   - `knowledgeBase page / stats / add / update / delete / graph / search / qaList / qaAdd`
3. 首批不开放：
   - `directoryTree`
   - `binaryUpload`
   - `permissionInheritance`
   - `richTextEditor`
   - `aiAnswer`
   - `employeePortal`

## 权限冻结结论

2. 部门经理：首批无后台入口。
3. 员工：首批无后台入口。

## 数据边界补充规则

1. 文件管理只返回元数据，不返回真实二进制、存储路径和存储凭证。
2. 知识库只返回标题、摘要、标签、关联文件、关联主题、重要度和查看次数等元数据，不返回正文全文、向量、提示词和模型配置。
3. 百问百答只返回问答元数据，不返回外部模型会话和外部知识源配置。
4. `relatedFiles` 首批只允许引用主题21文件台账 ID 或摘要关系，不承接真实文件正文。
