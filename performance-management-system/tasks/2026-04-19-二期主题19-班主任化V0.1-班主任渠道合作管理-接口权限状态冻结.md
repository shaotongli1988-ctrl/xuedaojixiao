## 已冻结资源边界

1. 资源名固定为：
   - `teacherInfo`
   - `teacherFollow`
   - `teacherCooperation`
   - `teacherClass`
   - `teacherDashboard`
   - `teacherTodo`
2. 首批接口固定为：
   - `teacherInfo page / info / add / update / assign / updateStatus`
   - `teacherFollow page / add`
   - `teacherCooperation mark`
   - `teacherClass page / info / add / update / delete`
   - `teacherDashboard summary`
   - `teacherTodo page`
3. 首批不开放：
   - `import`
   - `export`
   - `batchUploadAttachment`
   - `agent`
   - `performance`
   - `settlement`

## 权限冻结结论

1. 管理层 / 超级管理员：全量。
2. 部门负责人：仅本人部门树范围内全量管理，且可执行分配归属。
3. 部门员工：仅本人归属资源和关联班级可读写，无资源分配权限。
4. 只读账号：仅可查看授权范围内看板、资源列表 / 详情、待跟进和班级列表，无写权限。

## 合法流转

### 班主任资源状态

| 当前状态 | 动作 | 下一状态 | 允许角色 |
| --- | --- | --- | --- |
| 新增资源 | 保存 | `uncontacted` | 管理层 / 部门负责人 / 部门员工 |
| `uncontacted` | 新增首次跟进 | `contacted` | 管理层 / 部门负责人 / 部门员工 |
| `contacted` | 更新合作状态 | `negotiating` | 管理层 / 部门负责人 / 部门员工 |
| `contacted` / `negotiating` | 标记合作 | `partnered` | 管理层 / 部门负责人 / 部门员工 |
| `partnered` | 终止合作 | `terminated` | 管理层 / 部门负责人 |

### 班级状态

| 当前状态 | 动作 | 下一状态 | 允许角色 |
| --- | --- | --- | --- |
| 新增班级 | 保存 | `draft` | 管理层 / 部门负责人 / 部门员工 |
| `draft` | 更新班级状态 | `active` | 管理层 / 部门负责人 / 部门员工 |
| `active` | 关闭班级 | `closed` | 管理层 / 部门负责人 |

## 状态补充规则

1. 首批只允许对已 `partnered` 的班主任创建班级。
2. `terminated` 班主任不允许继续新建班级。
3. `closed` 班级不允许再编辑、删除或恢复。
4. 首批不引入代理、签约、结算、绩效等扩展状态。

## 数据边界补充规则

1. 资源列表 / 详情只返回班主任主数据、跟进摘要、合作摘要和班级摘要。
2. 只读账号查看联系方式时必须脱敏。
3. 看板和待跟进只返回聚合统计和本人 / 授权范围待办，不返回跨范围原始明细。
4. 首批不返回沟通截图原文、合作资料附件、代理体系数据、绩效分值和结算字段。
