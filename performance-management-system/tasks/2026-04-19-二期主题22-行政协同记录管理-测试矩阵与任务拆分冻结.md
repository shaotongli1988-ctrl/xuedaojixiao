## 最小测试矩阵

| 维度 | 最小验证点 |
| --- | --- |
| 正常路径 | `HR 管理员` 可在 5 个页面外壳下完成 `page / info / stats / add / update / delete` 主链，且 `moduleKey` 映射正确 |
| 异常路径 | 部门经理与员工访问主题22接口被拒绝；非法模块状态或非法字段值被拒绝；只读限制被拒绝 |
| 边界 | `status / keyword` 与各模块补充过滤字段可组合；空结果统计可正常返回；`extJson` 只接受轻量摘要结构 |

   - 各模块只读 / 删除限制

   - `build`
