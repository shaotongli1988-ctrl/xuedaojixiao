## 点页顺序

### 1. 登录

- 打开 `http://127.0.0.1:9028/#/login`
- 输入账号密码验证码并登录
- 预期：成功进入主界面，不停留在登录页

### 2. 招聘中心入口页

- 直接打开 `http://127.0.0.1:9028/#/performance/recruitment-center`
- 预期标题：`招聘中心入口页`
- 预期可见 6 张入口卡片：
  - `招聘计划`
  - `职位标准`
  - `简历池`
  - `面试管理`
  - `人才资产`
  - `录用管理`

### 3. 职位标准页

- 从入口页点击 `职位标准 -> 进入页面`
- 预期标题：`职位标准管理`
- 预期关键按钮：
  - `新增职位标准`
  - `新建招聘计划`
  - `新建简历`
- 关键跳转：
  - 点 `新建招聘计划`，应跳到招聘计划页
  - 返回后点 `新建简历`，应跳到简历池页

### 4. 招聘计划页

- 从入口页点击 `招聘计划 -> 进入页面`
- 预期标题：`招聘计划管理`
- 预期关键按钮：
  - `新建招聘计划`
  - `新建简历`
- 关键跳转：
  - 点 `新建简历`，应跳到简历池页
  - 如果列表里已有记录，打开详情后点 `查看职位标准` 或对应来源入口，应能回到职位标准页

### 5. 简历池页

- 从入口页点击 `简历池 -> 进入页面`
- 预期标题：`招聘简历池管理`
- 预期关键按钮：
  - `新建简历`
  - `发起面试`（仅在可发起记录上显示）
  - `查看招聘计划`
- 关键跳转：
  - 找一条可发起面试的简历，点 `发起面试`
  - 预期：跳到面试页，并自动打开新建出的面试详情
  - 找一条有关联招聘计划的记录，点 `查看招聘计划`
  - 预期：跳到招聘计划页，并打开对应详情

### 6. 面试管理页

- 从入口页点击 `面试管理 -> 进入页面`
- 预期标题：`招聘面试管理`
- 预期关键按钮：
  - `新建面试`
  - `查看简历`
  - `查看招聘计划`
  - `新建录用`
- 关键跳转：
  - 找一条来自简历池的面试，点 `查看简历`
  - 预期：跳到简历池页并打开对应详情
  - 点 `查看招聘计划`
  - 预期：跳到招聘计划页并打开对应详情
  - 找一条允许转录用的面试，点 `新建录用`
  - 预期：跳到录用页并打开对应详情或新建结果

### 7. 人才资产页

- 从入口页点击 `人才资产 -> 进入页面`
- 预期标题：`招聘人才资产`
- 预期关键按钮：
  - `新建人才资产`
  - `新建面试`
- 关键跳转：
  - 找一条人才资产记录，点 `新建面试`
  - 预期：跳到面试页
  - 预期：面试详情里来源摘要显示 `人才资产 #ID` 或同等来源信息

### 8. 录用管理页

- 从入口页点击 `录用管理 -> 进入页面`
- 预期标题：`录用管理`
- 预期关键按钮：
  - `新建录用`
  - `查看面试`
- 关键跳转：
  - 找一条来自面试的录用记录，点 `查看面试`
  - 预期：跳到面试页并打开对应详情

## 重点回归链路

- `职位标准 -> 招聘计划`
- `职位标准 -> 简历池`
- `招聘计划 -> 简历池`
- `简历池 -> 发起面试 -> 面试详情`
- `面试 -> 查看简历`
- `面试 -> 查看招聘计划`
- `面试 -> 新建录用 -> 录用详情`
- `人才资产 -> 新建面试 -> 面试详情`
- `录用 -> 查看面试 -> 面试详情`
